---
title: "HOW TO USE DIFFERENT THEME IN JEKYLL RUN WITHIN A LOCAL MACHINE"
layout: post
---

STEP 1 - Go to the official page of the theme. Check what its gem named.  
STEP 2 - add that gem to the gem file  
STEP 3 - run bundle update from command line  
STEP 4 - change the theme in the \_config.yml file  
STEP 5 - Go to the official git hub page of the theme. Download zip of the repository.  
STEP 6 - copy layout, \_includes, css, navigation files into the directory from where jekyll is running in the computer.
STEP 7 - Read the documentation to understand the theme.
